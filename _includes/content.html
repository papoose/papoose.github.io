About Papoose

Papoose is an open source implementation of OSGi R4 Service Platform licensed under the Apache License 2.0.

Features

  • OSGi Service Platform Core Specification R4
  • Configuration Admin Service Specification version 1.2
  • Event Admin Service Specification version 1.0
  • Http Service Specification version 1.2
  • Log Service Specification version 1.3

For more information, including setup and configuration instructions, bug reports or feature requests head over to GitHub.
